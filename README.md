# bywin 组件库

## 拉取代码
```
git clone https://github.com/wangxiaogang94/bywin-desgin.git
```
## 运行
```
cnpm i
npm start
```
## 打包 
```
npm run build:commonjs
npm run build:esm
npm run copy-css:esm
npm run copy-css:lib
```

## 发布

```
npm publish
```

## 开发

/src/components // 这里是开发的组件
/src/page // 这里是demo和文档
